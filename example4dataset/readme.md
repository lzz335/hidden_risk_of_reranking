# Dataset
Due to the size limitations of the anonymous repository, we only present a portion of the dataset we have constructed. The construction method for this dataset is detailed in Appendix C. We will release the complete dataset upon acceptance of this paper.
Additionally, the download links for the other open-source datasets used for evaluation in this paper are as follows:
- [Musique](https://huggingface.co/datasets/dgslibisey/MuSiQue)
- [NQ](https://github.com/google-research-datasets/natural-questions)
- [TriviaQA](https://huggingface.co/datasets/mandarjoshi/trivia_qa)
- [MultiHopRAG](https://github.com/yixuantt/MultiHop-RAG)
- [PQA](https://github.com/pubmedqa/pubmedqa)
